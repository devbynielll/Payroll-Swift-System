<h1 align="center">PayrollSwift</h1>
<p align="center">Payroll Management System</p>

---

## 📌 Overview

PayrollSwift is a desktop-based payroll management system developed to streamline employee payroll processing and administrative operations.

The system features a clean and organized user interface, secure authentication, and structured workflows designed for efficiency and ease of use.

---

## ✨ Features

- 🔐 User Authentication (Admin Access)
- 💰 Payroll Processing System
- 👥 Employee Management
- 📊 Organized and Structured Interface
- ⚙️ Administrative Controls
- 🧾 Payroll Record Handling

---

## 🛠️ Tech Stack

- Java
- NetBeans / Java GUI (Swing)
- Local Database (SQLite)

---

## ⚙️ Getting Started

### Requirements
- Java JDK installed
- NetBeans IDE (recommended)

### Steps to Run

1. Clone or download the repository  
2. Open the project in NetBeans  
3. Build the project  
4. Run the application
